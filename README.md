# CustomPong

It's a Custom Pong!

In Custom Pong we fused the classic mechanics of the well known Pong with many other retro (and not so retro) games to create a full new experience.

6 Challenging game modes including:

- Classic

- Bricks

- Pinball

- Flappy

- Warp

- Bubble
